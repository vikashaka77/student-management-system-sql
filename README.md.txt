 # Student Management System using MySQL

## Overview
A SQL-based Student Management System that stores student records, subjects, and marks while generating academic performance reports.

## Features
- Student record management
- Subject management
- Subject-wise marks storage
- CRUD operations
- Stored Procedures
- Average marks report
- Top scorer report
- Student count report

## Technologies Used
- MySQL
- SQL
- DBMS

## Database Schema
- Students
- Subjects
- Marks

## Reports Generated
- Total Students
- Average Marks
- Total Marks
- Top Scorer
